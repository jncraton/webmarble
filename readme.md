A simple interactive web-based marble run game.

[Demo](https://jncraton.github.io/jsmarble/)

## Usage

Select a tool from the palette and click on the canvas to place parts or drop marbles.

- Marble: Drops a red marble
- Ramp: Places a slanted platform
- Peg: Places a small circular obstacle
- Wall: Places a vertical barrier
- Trash: Clears all dynamic objects from the sandbox
